# Method-of-Four-Russians-for-bit-matrices
Advanced project

To use window application download file "Project.7z", unzip it and go to Project->WpfApp1->bin->Debug and run WpfApp1.exe.
To test console version download "ConsoleProject.7z", unzip it and go to ConsoleProject->Debug and run ConsoleApplication.exe.
There are 20 tests in console version. To test run console application and enter number between 1 and 20. If you enter 0, you can enter your own test.
There is a description to each test in ConsoleProject->ConsoleApplication2->Description.txt.
Tests are located in ConsoleProject->ColsoleApplication2->ConsoleTests.
Each test consists of 3 files: "input.txt", where are located input data, "output.txt", where is located method of four russians output and "correct_output.txt", where is located naive realization output.
In console version you can enter the number to run test, enter 0 to enter your own test in console and -1 to exit.
Аfter running the test, the result will be overwritten into the corresponding file "output.txt" and compared with the "correct_output.txt".
You can also run your test from file, create folder with name "Test21", for example, or with another number, write input and correct_output using one of my tests as an example and enter into program your test number.
There's also added the function of generating random test. To do it you jush have to enter 'g' and matrix size. Than it will be saved in folder 'Generated' with input and correct output. To run this test you have to enter 'G'.
