# Method-of-Four-Russians-for-bit-matrices
Advanced project

To use window application download file "Project.7z", unzip it and go to Project->WpfApp1->bin->Debug and run WpfApp1.exe.
To test console version download "ConsoleProject.7z", unzip it and go to ConsoleProject->Debug and run ConsoleApplication.exe.
There are 20 tests in console version. To test run console application and enter number between 1 and 20. If you enter 0, you can enter your own test.
There is a description to each test in ConsoleProject->ConsoleApplication2->Description.txt.
Tests are located in ConsoleProject->ColsoleApplication2->ConsoleTests.
You can also open project to see code and below in the comments there is a naive implementation code for comparison with my output.
